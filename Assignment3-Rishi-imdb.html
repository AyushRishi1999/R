Description about dataset

Predictors :-

 - Movie_Name - This column shows the name of each movie.
 - Release_Year - This column represents the year in which the moviw was released.
 - Duration - This column indicates the length in .
 - Metascore - This column represents the score received on a scale of 1-100.
 - Votes - This column represents the total number of votes given to a specified movie.
 - Genre - This represents all the genres covered by the specified movie.
 - Director - This column specifies the director of a specific movie.
 - Cast - This column specifies the cast in the specified movie.
 - Gross - This column signifies the gross earnings a movie made in a scale of million dollars

Outcome :-

 - IMDB_Rating - This column represents the average Rating a specified movie got over time.

Create a new observation 

x_test_data <- data.frame(
  Movie_Name = "Rocky",
  Release_Year = 1976,
  Duration = 120,
  Metascore = 70,
  Votes = 625924,
  Genre = "Drama, Sport",
  Director = "John G. Avildsen",
  Cast = "Sylvester Stallone",
  Gross = 117.24
)

new_observation_tbl_skim = partition(skim(x_test_data))
names(new_observation_tbl_skim)
## [1] "character" "numeric"
string_2_factor_names_new_observation <- new_observation_tbl_skim$character$skim_variable
rec_obj_new_observation <- recipe(~ ., data = x_test_data) |>
  step_string2factor(all_of(string_2_factor_names_new_observation)) |>
  step_impute_median(all_numeric()) |> # missing values in numeric columns
  step_impute_mode(all_nominal()) |> # missing values in factor columns
  prep()
new_observation_processed_tbl <- bake(rec_obj_new_observation, x_test_data)
new_application = new_observation_processed_tbl

h2o_exp_pdp <- model_profile(

Plot display for Partial dependence profiles

plot(h2o_exp_pdp, geom="profiles") +
ggtitle("Metascore")

Explaination of Partial-dependence Profiles

  • Partial dependence plots (PDP) are a valuable tool in understanding the relationship between a feature and the predictions made by a machine learning model. Here, we are using the h2o package to generate partial dependence profiles for the variable Metascore in our model.

Plot Summary

  • Decreasing Trend: The plot shows a steady increase in the predicted probability of Metascore prediction as the Metascore is increasing. It makes sense as Metascore affect the ratings directly.

  • Implication: The positive slope implies that, according to the model, higher Metascore will result to higher IMDB Rating.

  • Critical Temperature Thresholds: By observing graph we can say that rating threshold is approx 4 beyond which the negative impact on Metascore becomes more severe. This helps us in identify that till 4, it will not affect the Metascore in massive way.

plot(h2o_exp_shap) + ggtitle("SHAP explaination")

Explaination of SHAP

  • SHAP (SHapley Additive exPlanations) plots provide a visual representation of how individual features contribute to the prediction of a machine learning model. In this case, the values are provided in different percentiles (min, q1, median, mean, q3, max) of the SHAP distribution.

Summary

Votes = 625924 - It has Positive Impact on the models output on higher percentiles indicating that it contributes more on models output.

Release_Year = 1976 - It has Positive Impact on the models output.

Duration = 120 - It has Positive Impact on the models output.

Metascore = 70 - It has Positive Impact on the models output.

Movie_Name = Rocky - It has Positive Impact on the models output.

Genre = Drama, Sport - It has Negative Impact on the models output.

Director = John G. Avildsen - It has Negative Impact on the models output.

Cast = Sylvester Stallone - It has Negative Impact on the models output.

Gross = 117.24 - It has Negative Impact on the models output.

SHAP values offer valuable insights into the contribution of each feature to the models predictions, providing a deeper understanding of the decision-making process employed by the model.
